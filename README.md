Playground REPL to discover Node code snippets from Natural Language
Descriptions.

## Pre-requisites:
  npm<br>
   $> sudo apt-get install npm<br>
   $> sudo npm install -g npm@latest<br>
  node<br>
   $> sudo npm cache clean -f<br>
   $> sudo npm install -g n<br>
   $> sudo n stable<br>

## Instructions:

$> npm install<br>
$> python nql.py
